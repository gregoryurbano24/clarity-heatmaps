1. Purpose and success criteria

This guidebook is designed to be used directly by Roshni Hariharan and Johnny Goulbourne. Check off tasks as they are completed, add notes in the evidence log, and use the tracker tables to show completed work.

Project objective: implement and operationalize Microsoft Clarity heatmaps for the Mercedes-Benz Stadium website so the team can understand how fans use critical event-day pages and improve the site based on real behavior.

Fan impact

Help fans quickly find parking, directions, accessibility, bag policy, stadium maps, food and beverage, app, and event information.

Team impact

Create a repeatable review process that turns observations into prioritized website tasks.

Leadership impact

Maintain a visible record of what was reviewed, what was improved, and what still needs action.

16. Sources and notes

  • Microsoft Learn — Clarity Overview: Clarity is a behavioral analytics tool with session recordings, heatmaps, event/funnel tracking, and Copilot summarization; Microsoft also notes Clarity should not be used on websites/apps targeting users under 18.
  • Microsoft Learn — Heatmaps features: Heatmaps analyze aggregated user behavior; heatmaps require Clarity tracking code on pages; color scale shows hotter and colder interaction areas; heatmap data is retained up to 13 months; heatmaps are limited to up to 100,000 page views per heatmap.
  • Microsoft Learn — Segments in Clarity: Segments save groups of filters and can be applied to filters and custom tags.
  • Microsoft Learn — Custom tags in Clarity: Custom tags are customizable filters for recordings and heatmaps; the documented syntax is clarity("set", "key", "value");.
  • Mercedes-Benz Stadium website pages reviewed: Homepage, Parking and Transportation, Stadium Maps, Guidelines, Accessibility, All Vendors, Mobile Apps, MBS Express, Delta Fly-Through Lanes, Privacy Policy, and a FIFA World Cup 26 event detail page.
  • Internal AMBSE context: Digital platform documentation identifies mercedesbenzstadium.com as a managed web property and Webflow as the platform used to manage and maintain the MBS website.
